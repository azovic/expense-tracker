# 💸 Expense Tracker

A simple SQL-based personal finance tracker built with Python and SQLite.

## 🚀 Features
- Add categories
- Add payment methods
- Add expenses
- List all expenses
- Delete expenses
- View total spending
- Reports by category and payment method
- Category-based expense chart

## 🛠️ Technologies
- Python
- SQLite
- Matplotlib

## ▶️ How to Run

```bash
python3 main.py
